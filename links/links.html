Links no site

links externos:

Para colocar um link externo vinculado a um texto basta usar a tag "a".
Exenplo: click aqui para para ir para o youtube

É importante que você passa dois prametros para que o link abra em outra janela
e sua página não seja sobreposta. Os parametros são os seguintes: target = '_blank' e rel='external' .

links internos

Também é usado a tag 'a', entretanto, o parâmetro da ancora href vai receber o diretorio do arquivo html como valor

Acesse nossa página de notícias

como são págians internas, não há problemas em sobreposição

Links de dowloads

Para fazer um link de dowloads você deve passar para o parametro href o caminho do arquivao a ser baixado e além
disso dever passar o parametro dowloads e type onde esse tem valores definidos dependendo do tipo de arquivo -
esses valores definido podem ser encontrados nesse site
